# Ficha de Personagem - Sistema Elisio V2

Uma ficha de personagem interativa e completa para o sistema de RPG Elisio V2, com funcionalidades avançadas de salvamento e gerenciamento de dados.

## 🎯 Características

- **Interface moderna e responsiva**: Design limpo e profissional
- **Cálculos automáticos**: HP, MP, defesas e outros valores calculados automaticamente
- **Sistema de salvamento**: Salve múltiplas fichas no navegador
- **Import/Export**: Exporte suas fichas em JSON ou importe de outros arquivos
- **Auto-save**: Recuperação automática de dados em caso de fechamento acidental
- **Impressão otimizada**: Layout especial para impressão
- **Upload de imagem**: Adicione imagem do personagem

## 📋 Funcionalidades

### Informações Básicas
- Nome do personagem e jogador
- Raça, classe, nível e experiência
- Upload de imagem do personagem

### Sistema de Atributos
- 8 atributos principais: Força, Combate, Agilidade, Resistência, Intelecto, Carisma, Magia, Mana
- Skills específicas para cada atributo
- Valores editáveis e salvos automaticamente

### Vida, Mana e Defesa
- Cálculo automático de HP e MP baseado na classe
- Sistema de defesas: Defesa Padrão, Esquiva, Contra Ataque, Bloqueio
- Valores atuais e máximos editáveis

### Sistema de Salvamento
- **Salvar ficha**: Guarde múltiplas fichas com nomes personalizados
- **Carregar ficha**: Restaure fichas salvas anteriormente
- **Excluir ficha**: Remova fichas desnecessárias
- **Exportar JSON**: Baixe a ficha como arquivo JSON
- **Importar JSON**: Carregue fichas de arquivos externos
- **Auto-save**: Salvamento automático durante a edição

## 🚀 Como Usar

### Instalação
1. Baixe todos os arquivos (index.html, styles.css, script.js)
2. Abra o arquivo index.html em qualquer navegador moderno
3. Não é necessário servidor web - funciona localmente

### Criando uma Ficha
1. Preencha as informações básicas do personagem
2. Ajuste os valores dos atributos
3. Configure as skills de cada atributo
4. Adicione equipamentos e habilidades
5. Use o botão "Salvar Ficha" para guardar permanentemente

### Gerenciando Fichas
- **Para salvar**: Digite um nome na caixa "Nome da Ficha" e clique "Salvar Ficha"
- **Para carregar**: Selecione uma ficha na lista "Fichas Salvas" e clique "Carregar Ficha"
- **Para exportar**: Clique "Exportar JSON" para baixar um arquivo
- **Para importar**: Clique "Importar JSON" e selecione um arquivo válido

## 💾 Armazenamento de Dados

Os dados são salvos no localStorage do navegador:
- **Fichas salvas**: Persistem até serem excluídas manualmente
- **Auto-save**: Recupera automaticamente em caso de fechamento acidental
- **Exportação**: Permite backup externo em formato JSON
